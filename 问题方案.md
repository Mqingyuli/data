# 问题和解决方案

## 1.边缘模块的sx1280的频率问题

```c
/*
1.症状描述：
边缘节点的sx1280芯片1与sx1280芯片2接收频率设置为2424.5MHz时，芯片1数据接收异常，数据长度正常，但是校验码和数据错误；芯片2接收数据正常，包括长度、校验码和数据都吻合；
接入节点的sx1280芯片1与sx1280芯片2接收频率设置为2424.5MHz时，芯片1和2接收数据正常，包括长度、校验码和数据都吻合；
2.解决方法：
将边缘节点sx1280芯片1的接收频率设置为2424.55MHz~2424.94MHz之间的任意一个频点可以解决芯片1接收不正常的问题；同时测试sx1280芯片2的接收频率设置在2424.49MHz~2424.88MHz时，接收也都正常；芯片1和芯片2的频率如果设置为上述范围之外，数据接收异常，表现为数据长度正常，但是校验码和数据错误。可以看到，可以接收的频率的最大最小之差都是0.39Mhz。但是两个芯片的中心频率有偏差，偏差为0.06Mhz。
*/
```
